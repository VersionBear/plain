# Plain

Plain is a minimalist, local-first workspace for focused writing. It combines the speed of a web app with the privacy and longevity of local files.

## Features

- **Local-First**: Your notes stay on your device by default. No accounts, no servers, no tracking.
- **Markdown-Driven**: Full поддержка for CommonMark and GFM. What you see is what you get, but it stays pure Markdown.
- **Slash Commands**: Type `/` to quickly insert headings, tables, checklists, and dividers.
- **Folder Sync**: Connect a local folder to save notes as `.md` files that you can open in any other app (Obsidian, VS Code, etc.).
- **Smart Search**: Quickly find any note or tag with `Cmd/Ctrl + K`.
- **Image Support**: Drag and drop images directly into your notes.

## Why Plain?

We believe your thoughts shouldn't be trapped in a proprietary database. Plain is built on the philosophy that **privacy is a feature, not an afterthought.** By using standard formats and local storage, we ensure that your notes are always yours—even if Plain ceased to exist tomorrow.
