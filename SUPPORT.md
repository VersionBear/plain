# Plain Support

## Where to start

- Read the repository README for storage and browser-support details.
- If you are using browser-only storage, remember that notes stay in that
  browser on that device.
- If you need a second copy, connect a folder on a supported browser.

## Common issues

### My notes are missing on another device

Plain does not include built-in sync. Browser-only notes stay on the original
browser and device unless you connected a folder.

### Folder storage is unavailable

Some browsers and mobile platforms do not support the required file APIs.
Plain will continue working with browser-managed storage instead.

Common examples:

- iOS does not support folder storage in this app
- Mobile and PWA browser behavior can be less reliable than desktop browsers

### I need help with my Gumroad license

License verification happens with Gumroad. Keep your key available and try the
in-app re-check flow first. If verification still fails, gather the order
number and license key details before reporting the issue.

## Reporting bugs

Please open an issue on GitHub and include:

- Your browser and OS
- Whether you were using browser-only storage or a connected folder
- The exact action that failed
- Any error message shown by the app

GitHub issues: https://github.com/VersionBear/plain/issues
